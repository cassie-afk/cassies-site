Prototype Notes

这一版先把前端壳子搭稳，便于后续接 API 或 SQL 结果。

What This Version Already Solves

  • 固定了你要的 9 个 stage 顺序和来源归属。
  • 支持手工录入 stage volume，自动生成转化率和流失率。
  • 对跨系统口径不一致的异常做了即时提示。

Easy Next Step

  • ASA 侧补 `impression / taps / installs` 拉数接口。
  • Metabase 侧补剩余 stage 的 query 参数和字段映射。
  • 再加一个日期范围栏和 campaign selector 就能进入可用版本。
